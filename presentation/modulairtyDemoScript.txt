* Modularity Demo Script

* gradle createJavaProject
	* api-module
	* detect-module
	* ocr-module
	* pos-module
	* control-module
* parent build.gradle
	* allprojects apply plugin eclipse
	* allprojects apply plugin maven
	* allprojects apply plguin osgi
	* allprojects repositories
	* subprojects apply jar
	* osgi export !internal
* parent settings.gradle
	* include all modules

/sdks/gradle-2.4/bin/gradle  clean copyBundle buildImage
